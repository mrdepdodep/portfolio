# Mac Control — App (Русский)

Нативное macOS-приложение на SwiftUI с тёмным чат-интерфейсом для управления системой через текстовые команды.

## Требования

- macOS 12.0 или новее
- Xcode CLI tools / Swift компилятор (`xcode-select --install`)

## Сборка

```bash
cd app
bash build_app.sh
```

Создаёт `MacControl.app` внутри директории `app/`. Запустить:

```bash
open MacControl.app
```

## Структура

```
app/
├── Sources/MacControl/
│   ├── MacControlApp.swift   — точка входа приложения
│   ├── ContentView.swift     — корневой макет, хедер, поле ввода, подсказки
│   ├── MessageBubble.swift   — компоненты чат-пузырей
│   ├── CommandEngine.swift   — парсинг и выполнение команд, индекс приложений
│   ├── Models.swift          — модель сообщения
│   └── Theme.swift           — цветовая палитра
├── build_app.sh              — скрипт сборки (без Xcode)
└── run.sh                    — быстрый запуск через `swift run`
```

## Интерфейс

- **Хедер** — иконка терминала, пульсирующий статус-dot (зелёный = готово, янтарный = выполняется)
- **Чат** — сообщения пользователя с фиолетовым градиентом; ответы системы с левой акцентной полосой
- **Поле ввода** — monospaced шрифт с подсветкой фокуса; круглая кнопка отправки с тенью
- **Подсказки** — горизонтальный ряд чипов автодополнения на основе текущего ввода

## Команды

| Команда | Описание |
|---|---|
| `open <приложение>` | Открыть приложение |
| `close <приложение>` | Закрыть приложение |
| `volume <0-100>` | Установить громкость |
| `volume up\|down [шаг]` | Изменить громкость |
| `brightness <0-100>` | Установить яркость |
| `brightness up\|down [шаг]` | Изменить яркость |
| `mute` / `unmute` | Включить/выключить звук |
| `lock` | Заблокировать экран |
| `sleep` | Перевести дисплей в сон |
| `list` | Список всех проиндексированных приложений |
| `refresh` | Перестроить индекс приложений |
| `permissions` | Проверить / запросить разрешения macOS |
| `help` | Показать все команды |
| `exit` | Выйти из приложения |

## Разрешения

Некоторые команды требуют системных разрешений macOS:

- **Accessibility** — для эмуляции клавиш (громкость, яркость, блокировка экрана)
- **Automation → System Events** — для AppleScript-действий

Введите `permissions` в чате, чтобы инициировать запрос доступов.
