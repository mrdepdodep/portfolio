# Mac Control — CLI Script (Русский)

Терминальный Python-скрипт для управления macOS через текстовые команды: открытие/закрытие приложений, громкость, яркость, блокировка экрана, сон дисплея и многое другое.

## Требования

- macOS 12.0 или новее
- Python 3 (предустановлен на macOS)

## Запуск

```bash
cd script
python3 mac_control.py
```

## Команды

| Команда | Описание |
|---|---|
| `open <приложение>` | Открыть приложение |
| `close <приложение>` | Закрыть приложение |
| `volume <0-100>` | Установить громкость |
| `volume up\|down [шаг]` | Изменить громкость (шаг по умолч.: 10) |
| `brightness <0-100>` | Установить яркость |
| `brightness up\|down [шаг]` | Изменить яркость (шаг по умолч.: 10) |
| `mute` / `unmute` | Включить/выключить звук |
| `lock` | Заблокировать экран |
| `sleep` | Перевести дисплей в сон |
| `list` | Список всех проиндексированных приложений |
| `refresh` | Перестроить индекс приложений |
| `permissions` | Проверить / запросить разрешения macOS |
| `help` | Показать все команды |
| `exit` | Выйти из скрипта |

## Нечёткий поиск

Названия приложений и ключевые слова команд распознаются с нечётким поиском — допускаются опечатки и частичные названия (например, `open safri` откроет Safari).

## Разрешения

Некоторые действия требуют системных разрешений macOS:

- **Accessibility** — для эмуляции клавиш (громкость, яркость, блокировка экрана)
- **Automation → System Events** — для AppleScript-действий

Введите `permissions`, чтобы инициировать запрос разрешений через системный диалог.
