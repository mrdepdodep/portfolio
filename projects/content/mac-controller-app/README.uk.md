# Mac Control — App (Українська)

Нативний macOS-додаток на SwiftUI з темним чат-інтерфейсом для керування системою через текстові команди.

## Вимоги

- macOS 12.0 або новіша
- Xcode CLI tools / Swift компілятор (`xcode-select --install`)

## Збірка

```bash
cd app
bash build_app.sh
```

Створює `MacControl.app` всередині директорії `app/`. Запустити:

```bash
open MacControl.app
```

## Структура

```
app/
├── Sources/MacControl/
│   ├── MacControlApp.swift   — точка входу додатку
│   ├── ContentView.swift     — кореневий макет, хедер, поле вводу, підказки
│   ├── MessageBubble.swift   — компоненти чат-бульбашок
│   ├── CommandEngine.swift   — парсинг і виконання команд, індекс програм
│   ├── Models.swift          — модель повідомлення
│   └── Theme.swift           — кольорова палітра
├── build_app.sh              — скрипт збірки (без Xcode)
└── run.sh                    — швидкий запуск через `swift run`
```

## Інтерфейс

- **Хедер** — іконка терміналу, пульсуючий статус-dot (зелений = готово, бурштиновий = виконується)
- **Чат** — повідомлення користувача з фіолетовим градієнтом; відповіді системи з лівою акцентною смужкою
- **Поле вводу** — monospaced шрифт із підсвіткою фокусу; кругла кнопка відправки з тінню
- **Підказки** — горизонтальний ряд чіпів автодоповнення на основі поточного вводу

## Команди

| Команда | Опис |
|---|---|
| `open <програма>` | Відкрити програму |
| `close <програма>` | Закрити програму |
| `volume <0-100>` | Встановити гучність |
| `volume up\|down [крок]` | Змінити гучність |
| `brightness <0-100>` | Встановити яскравість |
| `brightness up\|down [крок]` | Змінити яскравість |
| `mute` / `unmute` | Вмикнути/вимкнути звук |
| `lock` | Заблокувати екран |
| `sleep` | Переключити дисплей у сон |
| `list` | Список усіх проіндексованих програм |
| `refresh` | Перебудувати індекс програм |
| `permissions` | Перевірити / запросити дозволи macOS |
| `help` | Показати всі команди |
| `exit` | Вийти з додатку |

## Дозволи

Деякі команди потребують системних дозволів macOS:

- **Accessibility** — для емуляції клавіш (гучність, яскравість, блокування екрана)
- **Automation → System Events** — для AppleScript-дій

Введіть `permissions` у чаті, щоб ініціювати запит доступів.
