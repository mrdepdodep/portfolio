# Mac Control — CLI Script (Українська)

Термінальний Python-скрипт для керування macOS через текстові команди: відкриття/закриття програм, гучність, яскравість, блокування екрана, сон дисплея та інше.

## Вимоги

- macOS 12.0 або новіша
- Python 3 (вже встановлений на macOS)

## Запуск

```bash
cd script
python3 mac_control.py
```

## Команди

| Команда | Опис |
|---|---|
| `open <програма>` | Відкрити програму |
| `close <програма>` | Закрити програму |
| `volume <0-100>` | Встановити гучність |
| `volume up\|down [крок]` | Змінити гучність (крок за замовч.: 10) |
| `brightness <0-100>` | Встановити яскравість |
| `brightness up\|down [крок]` | Змінити яскравість (крок за замовч.: 10) |
| `mute` / `unmute` | Вмикнути/вимкнути звук |
| `lock` | Заблокувати екран |
| `sleep` | Переключити дисплей у сон |
| `list` | Список усіх проіндексованих програм |
| `refresh` | Перебудувати індекс програм |
| `permissions` | Перевірити / запросити дозволи macOS |
| `help` | Показати всі команди |
| `exit` | Вийти зі скрипту |

## Нечіткий пошук

Назви програм і ключові слова команд розпізнаються з нечітким пошуком — допускаються помилки та часткові назви (наприклад, `open safri` відкриє Safari).

## Дозволи

Деякі дії потребують системних дозволів macOS:

- **Accessibility** — для емуляції клавіш (гучність, яскравість, блокування екрана)
- **Automation → System Events** — для AppleScript-дій

Введіть `permissions`, щоб ініціювати запит дозволів через системний діалог.
